# Archive

This folder contains deprecated or superseded materials that were kept for safety instead of deleted.

Structure:

- `decks_old/`
  - older defense decks, optional insert decks, and old visual references
- `demo_old/`
  - reserved for older demo packages if additional historical copies need to be parked here
- `reports_old/`
  - older notes, audits, rebuild reports, and workspace report history
- `generated_intermediate/`
  - reserved for non-current generated intermediates
- `packages_old/`
  - superseded convenience bundles and older package-level archives

Nothing in this folder is the official current entrypoint.
